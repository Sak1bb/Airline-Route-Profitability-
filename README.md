# Airline-Route-Profitability-
This project analyses airline route profitability using a realistic aviation dataset. It explores the impact of revenue, operational costs, demand, aircraft type, and seasonality on route performance, providing data-driven insights to support route planning, cost optimisation, and profitability.
# ✈️ Airline Route Profitability Analysis

## 📖 Project Overview
This project analyses airline route profitability using a realistic aviation dataset containing 7,974 commercial flights operating from Dubai International Airport (DXB) during 2024. The objective was to identify the operational and financial factors influencing route profitability and apply business intelligence and data mining techniques to generate actionable insights.

The project explores airline performance through data cleaning, exploratory data analysis (EDA), interactive visualisation, and predictive modelling. It demonstrates how data analytics can support route planning, cost optimisation, aircraft allocation, and strategic business decision-making.

---

## 🎯 Objectives
- Analyse operational and financial performance of airline routes.
- Identify the key drivers of route profitability.
- Explore patterns across seasons, aircraft types, demand levels, and route distances.
- Build predictive models for profitability.
- Generate business recommendations using data-driven insights.

---

## 📊 Dataset
- **Source:** Simulated Airline Route Profitability Dataset
- **Flights:** 7,974
- **Routes:** 30 International Destinations
- **Airport:** Dubai International Airport (DXB)
- **Variables:** 33 Features
- **Categories:** Revenue, Costs, Aircraft Type, Season, Demand Level, Distance Tier and Profitability
